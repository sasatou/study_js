# JavaScript

## 変数を宣言する

### 宣言
* var
* let(ES6)
* const(ES6)

**構文**

```javascript:title
宣言 変数名 代入 初期値
var hoge = 10;
```
宣言だけの場合
```javascript:title
let hoge;
```
カンマで区切って複数宣言
```javascript:title
let hoge, hoge2;
```
宣言時に初期値を代入
```javascript:title
let hoge = 'ハローワールド！';
```
***！let命令は同名の変数を許可しないので注意***
